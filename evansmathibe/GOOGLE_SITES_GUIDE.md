# Google Sites Setup Guide for Evans Mathibe

## Getting Started with Google Sites
1. Go to [sites.google.com](https://sites.google.com)
2. Sign in with your Google account
3. Click "Create" to start a new site

## Recommended Site Structure

### Pages to Create
1. **Home** - Main landing page with hero, about preview, services
2. **About** - Full about section, timeline/experience, stats
3. **Services** - Detailed services offered
4. **Testimonials** - Client testimonials
5. **Gallery** - Image gallery
6. **Contact** - Contact form and information

## Google Sites Features to Use

### Built-in Features
| Feature | How to Use |
|---------|------------|
| **Google Forms** | Embed contact form - go to forms.google.com, create form, click "Send" > "Embed" |
| **Google Calendar** | Embed availability/meeting calendar via Insert > Calendar |
| **YouTube Videos** | Insert > Videos to add promotional videos |
| **Google Maps** | Insert > Maps to show your location |
| **Google Docs/Sheets** | Embed resumes, price lists, portfolios |
| **File Downloads** | Upload PDFs for download (brochures, resumes) |

### Customization Options
- **Themes** - Choose from built-in themes in the right panel
- **Colors** - Customize site colors to match your brand
- **Fonts** - Select from Google Fonts
- **Header** - Add logo and site name
- **Navigation** - Drag to reorder pages
- **Footer** - Add contact info and social links

### Embed Code (for advanced features)
Use Insert > Embed > Embed code for:
- Social media feeds
- Contact forms (Typeform, Formspree)
- Chat widgets
- Analytics
